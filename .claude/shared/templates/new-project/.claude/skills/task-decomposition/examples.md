## Example 1: Logging Standards Integration

**Context**: Add mandatory structured logging to all code produced by development system

**Input**: User request to integrate logging standards across 6 files + new guide

**What happened**:
- Identified 3 independent waves: (1) create guide, (2) update 5 files in parallel, (3) sync to template
- Wave analysis: guide must exist before files reference it → Wave 1 blocked Wave 2
- 5 file updates were independent → all in Wave 2 as parallel tasks

**Output**: 3-wave plan with 7 tasks, executed via Agent Teams (5 parallel agents in Wave 2)

**Learning**: Template changes often have a "create first, then reference" dependency pattern
