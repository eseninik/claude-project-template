Можно обойти интерактивный onboard через `daemon install` если конфиг уже готов. Gateway требует 1-2 минуты на загрузку (1.5GB RAM).

**Gotchas:** `execFileUtf8` в Node.js переписывает stderr своим сообщением об ошибке — нужно проверять реальный stdout. Systemctl возвращает разные exit коды (0=enabled, 1=disabled, 4=not-found) — нужна правильная обработка каждого.
