## Интеграционные правила

### При создании заказа:
1. Проверить наличие товаров
2. Заблокировать товары в резерве
3. Создать платеж в платежной системе
4. Отправить уведомление пользователю

### При отмене заказа:
1. Разблокировать товары
2. Инициировать возврат (если оплачен)
3. Отправить уведомление
4. Обновить статистику
