# Список фич

## Назначение
Полный инвентарь фич проекта с приоритетами, статусами и зависимостями.

---

## Статусы

- `planned` - Запланировано, не начато
- `in-progress` - В разработке
- `done` - Готово, в продакшене
- `on-hold` - Приостановлено

## Приоритеты

- `P0` - Критично, без этого не запускаемся
- `P1` - Важно, нужно в первой версии
- `P2` - Желательно, можно отложить
- `P3` - Nice to have
